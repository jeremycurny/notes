# Bower

## Manage packages

### Init

```
bower init
```

### Example : Installing jquery

```
bower install jquery --save
```

## bower.json examples

```
{
  "name": "vendor"
}
```

```
{
  "name": "vendor",
  "dependencies": {
    "jquery": "^3.1.1",
    "bootstrap": "^3.3.7" 
  }
  "private": true
}
```
